SixTrack is a single particle 6D symplectic tracking code optimized for long term tracking in high energy rings.

It is mainly used for the LHC for dynamic aperture studies, tune optimization, collimation studies.

The core is single executable orginated by Fortran pre-processed sources that can be accessed from the Source pages. The code is licensed under LGPLv2.1 under the CERN copyright.

Manual and reference papers are available in the Documentation pages.

A developer manual is being written in the Wiki pages.

A Linux environment, SixDesk, is available in the Source pages for preparing, running, and analysing dynamic aperture studies for the LHC. It can use the CERN LSF and LHC@Home computer resources.

The former website is available here.
