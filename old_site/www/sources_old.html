SixTrack

The sixtrack sources are available in the SVN repository. Files can be checked out with 

 svn co svn+ssh://svn.cern.ch/reps/SixTrack
or 
 svn co http://svn.cern.ch/guest/SixTrack

SixDesk LHC Tracking environment

The sixdesk sources are available in the SVN repository. Files can be checked out with 

 svn co svn+ssh://svn.cern.ch/reps/sixdesk
or 
 svn co  http://svn.cern.ch/guest/sixdesk

Developements

Specific on going developements are hosted on the GitHub SixTrack page and they will normally merged with official repository as soon as the code is stable.
  • SixDeskDB: SixDesk environment implemented in Python/SQLite
  • SixTrackLib: C library implementing SixTrack models.
  • DYNK: SixTrack fork implementing dynamic kicks.
